## Prerequisites
1. NodeJS and npm
2. gulp


## Installation

### Setup
1. `git clone https://github.com/csgo-league/csgo-league-bot`
2. `npm i`
3. `npm i -g gulp`
4. Edit config.json
5. Run the bot, when it has created a text and a voice channel, restart it.


### Launch
1. `gulp`
