## Prerequisites
1. NodeJS and npm
2. gulp


## Installation

### Setup
1. `git clone https://github.com/csgo-league/csgo-league-bot`
2. `npm i`
3. `npm i -g gulp`
4. Edit config.json


### Launch
1. `gulp`
